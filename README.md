## Description

J'ai décidé d'aborder une approche vers un restaurant qui s'est insipiré de la vie d'un personnage d'anime. Oguri Cap. C'est un personnage connu pour manger beaucoup, donc j'ai décidé de jouer avec.

## Choix techniques

**Palette de couleurs :**

#00ffff
rgba(69, 63, 63, 1)
rgba(255, 255, 255, 1)
rgb(95, 144, 208)

**Police utilisée :**

- Playwrite NZ Basic

- Orbitron

**Décisions CSS notables :**

- J'ai décidé d'aller avec un style très robotique, car je trouve qu'un thème de cyberpunk mélanger avec le «« monstre oguri cap »» donne son style. Le dégradé du blanc vers le noir reflète les couleurs d'oguri cap. Cela est pourquoi j'ai décidé de faire un background en ses couleurs.
